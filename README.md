# Portfolio — Yanis Benmimoun

Portfolio personnel présentant mon parcours, mes compétences et mes projets en développement web et cybersécurité.

## 🛠️ Technologies

- **HTML5 / CSS3 / JavaScript**
- **Google Fonts** — Inter, JetBrains Mono
- **Font Awesome** — Icônes
- **Web Crypto API** — Utilisée dans le Hash Generator

## 📂 Structure du projet

## 🚀 Projets intégrés

| Projet | Description | Technologies |
|--------|-------------|--------------|
| **Générateur de Mots de Passe** | Génération de mots de passe sécurisés avec indicateur de force et options personnalisables | HTML, CSS, JS |
| **Chiffrement César** | Chiffrement et déchiffrement avec table de correspondance visuelle en temps réel | HTML, CSS, JS |
| **Hash Generator** | Calcul d'empreintes SHA-256, SHA-512, SHA-1 et MD5 avec copie instantanée | JS, Web Crypto API |
| **Mon Salon** | Site vitrine responsive d'un salon de coiffure fictif | HTML, CSS, JS |

## ⚡ Lancement

Ouvrir `index.html` dans un navigateur. Aucune dépendance à installer, tout fonctionne en HTML/CSS/JS pur.
